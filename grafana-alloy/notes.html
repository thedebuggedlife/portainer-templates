Application Data

The container is pre-configured to store persistent data for Alloy under /srv/appdata/alloy.

You may change the location by expanding Show advanced options below, then look under Volume mapping.

To create the parent application data folder:

sudo mkdir -p /srv/appdata/alloy
sudo chown $USER /srv/appdata/alloy
touch /srv/appdata/alloy/config.alloy

Alloy Configuration

You need to configure Alloy according to your monitoring requirements before launching the Alloy service. By default, the configuration file is expected to be in the path /srv/appdata/alloy/config.alloy`.
