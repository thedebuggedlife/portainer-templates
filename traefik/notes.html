Application Data

The container is pre-configured to store persistent data for LLDAP under /srv/appdata/traefik.

You may change the location by expanding Show advanced options below, then look under Volume mapping.

To create the parent application data folder:

sudo mkdir -p /srv/appdata/traefik
sudo wget -qO /srv/appdata/traefik/traefik.yml https://raw.githubusercontent.com/thedebuggedlife/portainer-templates/refs/heads/main/traefik/traefik.yml
sudo chown -R nobody:docker /srv/appdata/traefik
