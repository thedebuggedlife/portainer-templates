Application Data

The container is pre-configured to store persistent data for LLDAP under /srv/appdata/lldap.

You may change the location by expanding Show advanced options below, then look under Volume mapping.

To create the parent application data folder:

sudo mkdir -p /srv/appdata
sudo chown $USER /srv/appdata

Generate Secrets

LLDAP requires pair of secrets for JWT and KEY seed. These can be generated by running the following: 

wget -qO- https://raw.githubusercontent.com/lldap/lldap/refs/heads/main/generate_secrets.sh | bash

Then copy and paste the secrets from the output to the respective variables in the template.

Domain Name Format

Use the following format for the Base Domain Name template variable: dc=example,dc=com if your domain is example.com
