Application data MUST be set up BEFORE deploying this stack using the following: 

wget -qO- https://thedebuggedlife.github.io/portainer-templates/appdata/linux-monitoring.sh | bash

You can enable or disable individual performance monitors with the COMPOSE_PROFILES variable. See the below for details:

  • node: Monitors CPU, RAM, temperature and other machine-level metrics
  • process: Monitors resources used by individual processes
  • smartctl: Monitors SMART metrics for SDD and HDD devices
  • cadvisor: Monitors resources used by docker containers

See the template README or the related blog post for more info.
