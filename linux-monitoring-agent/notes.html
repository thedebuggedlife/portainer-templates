Application data MUST be set up BEFORE deploying this stack! 

# Update if your appdata path is different
appdata=/srv/appdata

# Download appdata files
wget https://thedebuggedlife.github.io/portainer-templates/appdata/linux-monitoring-agent.zip

# Unpack into local appdata folder
unzip -n linux-monitoring-agent.zip -d $appdata/

# Setup proper permissions
sudo chown -R nobody:docker $appdata/process-exporter
sudo chown -R nobody:docker $appdata/prometheus

# Replace REMOTE_HOST in the following file
read -n 1 -s -r -p "IMPORTANT: Replace REMOTE_HOST in the file $appdata/prometheus/prometheus.yml"
nano $appdata/prometheus/prometheus.yml

See the template README or the related blog post for more info.
