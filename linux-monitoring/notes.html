Application data MUST be set up BEFORE deploying this stack using the following: 

wget -qO- https://thedebuggedlife.github.io/portainer-templates/appdata/linux-monitoring.sh | bash

You can enable or disable individual performance monitors with the 

COMPOSE_PROFILES
variable. See the following table for details:

Profile Monitors Notes
node CPU, RAM, temperature and other machine-level metrics
process Resources used by individual processes
smartctl SMART metrics for SDD and HDD devices Not supported on arm/arm64 devices
cadvisor Resources used by docker containers

See the template README or the related blog post for more info.
